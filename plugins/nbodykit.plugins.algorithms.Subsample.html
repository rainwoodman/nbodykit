create a subsample from a DataSource, and evaluate density (1 + delta) smoothed at the given scale

datasource the DataSource to read; run nbkit.py –list-datasources for all options Nmesh the size of FFT mesh for painting
seed the random seed (default: 12345) ratio the fraction of particles to keep (default: 0.01) smoothing the smoothing length in distance units. It has to be greater than the mesh resolution. Otherwise the code will die. Default is the mesh resolution. format the format of the output (default: hdf5) comm the global MPI communicator
