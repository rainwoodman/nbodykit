plugins.algorithms.BianchiFFTPower module¶

class plugins.algorithms.BianchiFFTPower.BianchiPowerAlgorithm(data, randoms, Nmesh, max_ell, dk=None, kmin=0.0, BoxSize=None, BoxPad=0.02, compute_fkp_weights=False, P0_fkp=None, nbar=None, fsky=None, quiet=False, factor_hexadecapole=False, keep_cache=False)[source]¶

Bases: nbodykit.extensionpoints.Algorithm

Algorithm to compute the power spectrum multipoles using FFTs for a data survey with non-trivial geometry

The algorithm used to compute the multipoles is detailed in Bianchi et al. 2015 (http://adsabs.harvard.edu/abs/2015MNRAS.453L..11B)
