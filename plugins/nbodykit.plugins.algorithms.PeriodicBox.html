class nbodykit.plugins.algorithms.PeriodicBox.FFTCorrelationAlgorithm(*args, **kwargs)¶

Bases: nbodykit.extensionpoints.Algorithm

Algorithm to compute the 1d or 2d correlation function and multipoles in a periodic box. This FFTs the measured power spectrum to compute the correlation function

Methods

__init__(*args, **kwargs)¶

correlation spectrum calculator via FFT in a periodic box

mode compute the power as a function of k or k and mu Nmesh the number of cells in the gridded mesh field first data field; a tuple of (DataSource, Painter, Transfer)

DataSource the 1st DataSource; run nbkit.py –list-datasources for all options Painter the 1st Painter; run nbkit.py –list-painters for all options Transfer the 1st Transfer chain; run nbkit.py –list-transfers for all options
other the other data field; a tuple of (DataSource, Painter, Transfer)
DataSource the 2nd DataSource; run nbkit.py –list-datasources for all options Painter the 2nd Painter; run nbkit.py –list-painters for all options Transfer the 2nd Transfer chain; run nbkit.py –list-transfers for all options

los the line-of-sight direction – the angle mu is defined with respect to (default: z) Nmu the number of mu bins to use from mu=[0,1]; if mode = 1d, then Nmu is set to 1 (default: 5) dk the spacing of k bins to use; if not provided, the fundamental mode of the box is used kmin the edge of the first k bin to use; default is 0 (default: 0.0) quiet silence the logging output (default: False) poles if specified, also compute these multipoles from P(k,mu) (default: []) paintbrush the density assignment kernel to use when painting; CIC (2nd order) or TSC (3rd order) (default: cic) comm the global MPI communicator
