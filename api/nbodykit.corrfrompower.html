nbodykit.corrfrompower module¶

nbodykit.corrfrompower.corrfrompower(K, P, R, res=100000)[source]¶

calculate correlation function from power spectrum, P is 1d powerspectrum. if R is not None, estimate at those points . returns R, xi(R) internally this does a numerical integral with the trapz rule for the radial direction of the fourier transformation, with a gaussian damping kernel (learned from Xiaoying) the nan points of P is skipped. (usually when K==0, P is nan) input power spectrum is assumed to have the gadget convention, AKA, normalized to (2 * pi) ** -3 times sigma_8. The integral can also be done on log K instead of K if logscale is True.
