nbodykit.utils.taskmanager.tasks_parser(value)[source]¶

Given a string of the format key tasks, split the string and then try to parse the tasks, by first trying to evaluate it, and then simply splitting it and interpreting the results as the tasks.

The general use cases are:

  1. “box: range(2)” -> key = box, tasks = [0, 1]
  2. “box: [‘A’, ‘B’ ‘C’]” -> key = box, tasks = [‘A’, ‘B’, ‘C’]
