
Maintainer's corner
===================

Dependency of tests
-------------------

- `pytest`
- `pytest-pipeline`
- `pytest-cov`
- `coveralls`

To run the tests use

.. code::

    py.test nbodykit


or a specific test

.. code::

    py.test nbodykit/test/test_batch.py::TestStdin


Steps to tag a release
----------------------

1. Bump the version, in nbodykit/__init__.py, removing `.dev0` postfix
2. Add a git commit, add the version tag, push to master.
3. python setup.py sdist
4. Upload the .tar.gz file in dist/ to pip (optional?)
5. Bump the version, in nbodykit/__init__.py, raise the version number,
   add `.dev0` postfix
6. Add a git commit, push to master.
